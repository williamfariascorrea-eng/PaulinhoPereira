# Paulinho Pereira — Pré-candidato a Deputado Estadual RS 2026

## Site oficial de campanha

Site profissional para o pré-candidato **Paulinho Pereira** (Podemos). Três mandatos de vereador em São Lourenço do Sul, agora rumo à Assembleia Legislativa do Rio Grande do Sul em 2026.

## Estrutura do Projeto

```
projeto-paulinho/

```
